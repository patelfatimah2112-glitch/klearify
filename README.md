🟦 Klearify
Learn Anything. Your Way.

An AI-powered personalized learning platform that explains any topic through the lens of what you already love.

🚀 Live Demo
🔗 Try Klearify Live → https://klearify.vercel.app

💡 The Problem
Education is one-size-fits-all.
A cricket fan, a foodie, and a gamer all learn differently — yet every textbook explains gravity the same way. Students disengage because examples feel irrelevant to their world.
Klearify fixes this.

✨ How It Works
Step           Action
1️⃣              Enter any topic you want to understand
2️⃣              Select your interest(cricket, cooking, gaming...)
3️⃣              Choose your language
4️⃣              Pick your explanation level (Super Simple / School level /                     College(university) level / Expert level)
5️⃣              Hit "Explain It My Way" — get a personalized explanation                       instantly ⚡


🔥 Features

🔐 User Authentication — Secure login & signup
📜 Explanation History — All your past explanations saved
📄 PDF Download — Download any explanation as a formatted PDF
🎟️ Daily Credits System — Scalable, monetization-ready architecture
⚡ Groq API — Lightning-fast AI responses
🌍 Multi-language Support — Learn in your own language

🛠️ Tech Stack
Frontend   →  HTML, CSS
Backend    →  Python
AI Engine  →  Groq API
Database   →  Supabase
Hosting    →  Vercel
